# OEOE Agent DAO 🚀

> 无项目方、纯社区自治的 OEOE 冲奖方案

基于 X Layer 的 MEME 社区自治 AI 代理协作网络

## 📋 项目概述

- **项目名称**: OEOE Agent DAO
- **参赛赛道**: X Layer 黑客松
- **核心叙事**: 无项目方、无团队，由 OEOE 持币者自发组建 DAO，共同开发 AI 代理工具
- **技术栈**: X Layer + x402 支付 + Onchain OS + AI 代理

## 🎯 核心功能

### 1. 代理支付模块 (x402)
- OEOE 付费社群、AI 内容打赏、社区订阅服务
- 链上微支付，支持 OKB/USDC/OEOE
- Gas≈0，用户无需注册，直接链上结算

### 2. AI 代理自治模块 (Onchain OS)
- DAO 自动投票、分红分发、社区公告推送
- 自治机器人：自动统计投票、分发分红、发布动态
- 多代理协作架构

### 3. 社区影响力模块
- X Layer 活跃数据统计、贡献排行
- 可视化图表展示社区热度

## 📁 项目结构

## 🚀 快速开始

### 前置要求
- Node.js 18+
- X Layer 钱包
- OKX Onchain Gateway API

### 安装
```bash
npm install
```

### 运行
```bash
npm run dev
```
